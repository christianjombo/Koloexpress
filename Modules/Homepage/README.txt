To add a new Addon to your MarketplaceKit installation, follow these basic steps:

------------------------------
Automatic installation
------------------------------

1. Download the Addon archive
2. Go to the "Addon" section in the admin panel
3. Click "Upload Zip" and upload the zip
4. Activate the addon and configure the settings


------------------------------
Manual installation
------------------------------

1. Download the Addon archive and extract the files it contains to the "Modules/" folder
2. Go to the "Addon" section in the admin panel
3. Activate the addon and configure the settings
